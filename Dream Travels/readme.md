## 🧭 Overview

**Dream Travels and Resorts** is a modern one-page travel website built with **HTML5**, **Bootstrap 5**, and **custom CSS**. It showcases travel destinations, tour packages, and includes a **contact form** that displays a confirmation message upon submission.

The design focuses on:
- High-impact visual presentation
- Seamless scrolling navigation
- Mobile responsiveness
- Interactive form feedback

---

## 🎯 Project Requirements

The project was built based on the following specifications:

- Central **hero section** with customizable background image
- Easy-to-switch background images from a local `./pictures` folder
- Structured layout with:
  - Navbar
  - Hero banner
  - Destinations section
  - Accordion-style tour packages
  - Centered contact form
- Contact form should:
  - Be centered and responsive
  - Display a message upon submission:  
    _“Thank you for contacting us, we respond in 48 hours.”_
- Button from hero section scrolls smoothly to contact form
- Mobile-first design

## 🔧 How to Use

1. **Clone this Repository:**
   ```bash
   git clone https://github.com/your-username/dream-travels.git
   cd dream-travels
````

2. **Place your images** in the `/pictures` folder. Update the `hero` section's background URL if needed:

   ```css
   .hero {
     background: url('./pictures/your-image.png') no-repeat center center;
     ...
   }
   ```

3. **Open `index.html`** in your browser:

   ```bash
   open index.html
   ```

4. Click the **"Start Your Journey"** button to scroll to the **Contact** section.

## ✅ Contact Form Logic

The contact form is designed to:

* Capture Name, Email, Destination, and Message
* Display a **success message** using JavaScript on submit without a page reload:

  > “Thank you for contacting us, we respond in 48 hours.”
